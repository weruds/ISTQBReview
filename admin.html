⚠ Data Management Tools

Clear Sessions for a Specific User
Removes all session records (and leaderboard entries) for the selected user.
Clear All Sessions by Mode
Deletes all session records matching the selected mode from both the session log and leaderboard index.
⛔ Clear ALL Leaderboard Scores
Wipes the entire leaderboard index. Session records are kept. This cannot be undone.
⛔ Clear ALL Session Records
Wipes the full session log AND the leaderboard index. User profiles are kept. This cannot be undone.
⛔ Delete a User Profile
Removes the user's profile (avatar, Entry ID, streak). Their score records are kept unless cleared separately.
